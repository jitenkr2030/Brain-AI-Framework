### **ğŸš€ Quick Start - 5 Minutes to Production**
```python
# Install the SDK
pip install brain-ai-sdk

# Use the framework
from brain_ai import BrainAISDK, BrainAIConfig

config = BrainAIConfig("http://localhost:8000")
sdk = BrainAISDK(config)

# Store a memory (persists forever)
memory_id = await sdk.store_memory(
    content="Customer prefers email support over chat",
    type="episodic"
)

# Search for similar memories (sub-millisecond)
results = await sdk.search_memories("customer preferences", limit=5)

# Ask questions (uses all stored knowledge)
answer = await sdk.reason("How should we handle this customer?")
```
